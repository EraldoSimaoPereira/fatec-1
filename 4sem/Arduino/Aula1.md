# Arduino UNO

### Placa contém:
- Micro controlador ATMega328
- Estabilizador de voltagem 5v
	- Entrada de 7v a 12v
	- Saída de 5v
- Conectores de acesso aos pinos
- Suporte USB
	- Carregar programas
	- Comunicação com o computador

### Micro controlador
- CPU roda a 16 MHz
- Memória
- Flash - 32kb (4k bootloader)
