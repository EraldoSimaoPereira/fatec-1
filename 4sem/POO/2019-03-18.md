# Copy constructor
É usado para criar um objeto copiando os valores de atributos de outro objeto já existente.
Tem como único parâmetroo objeto que será copiado.

```java
	// Copy constructor
    public Ponto(Ponto pt)
    {
	    
       this(pt.x, pt.y);
    }
```
